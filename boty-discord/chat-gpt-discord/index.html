Integracja CHAT-GPT w discord

Bot może generować NIEZROZUMIAŁE i NIEPOPRAWNE odpowiedzi. Często psuje się gdy pare osób zadaje pytania jednocześnie

    Jest testowy bot który używa api chat-gpt aby prowadzić konwersacje z użytkownikiem w kanale tekstowym.
    Komenty oraz konwersacja jest prowadzona poprzez kanał w aplikacji discord.

    W pliku należy dodać klucz api (openai) oraz token bota (discord).
